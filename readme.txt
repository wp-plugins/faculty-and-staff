The all-purpose faculty and staff list.

== Description ==

Plugin adds a new Post Type, `Faculty and Staff` which is then placed on any page where the proper shortcode is installed. This plugin is intended for anyone wishing to give a summary of information on each person involved in their business or organization, as well as provide a detailed sheet for each user.

= Features =

* Short code allows to be placed on any page
* jQuery UI enabled whitebox
* Simple to use
* Responsive *mostly, in progress*

== Installation ==

This section describes how to install and use the plugin.

1. Install through the `Plugins` menu, using the `Add New` button. (or upload the entire `hfd-facultystaff` folder to the `/wp-content/plugins/` directory)
2. Activate the plugin.
3. Place shortcode `[hfd-facultystaff]` on desired page.
4. Go to `Faculty and Staff` -> `Add New`.
5. Enter `Title`, *Full Content* into the `content area`, and the *Excerpt* into the `excerpt area`.
6. Click `Update`.
7. Repeat steps *4 through 6* for each person you'd like to add.
8. Verify content on page.
